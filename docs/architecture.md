## 2. 技术栈

### 后端
- **框架**: FastAPI (Python 3.11+)
- **ORM**: SQLAlchemy 2.0
- **验证**: Pydantic v2
- **认证**: JWT + OAuth2

### 前端
- **框架**: React 18 + Next.js 14
- **UI**: Tailwind CSS + shadcn/ui
- **状态**: Zustand
- **HTTP**: Axios / SWR

### 数据库
- **主库**: MySQL 8.0 (InnoDB)
- **缓存**: Redis 7.x
- **搜索**: Elasticsearch 8.x (可选)

### AI 服务
- **主模型**: 通义千问-Max / GPT-4o
- ** embedding**: text-embedding-3-small

### 基础设施
- **容器**: Docker + Docker Compose
- **CI/CD**: GitHub Actions
- **监控**: Prometheus + Grafana

## 3. 核心模块设计

### 3.1 用户模块 (User Service)

```
职责: 用户注册、登录、个人信息管理

API:
- POST /api/v1/auth/register - 注册
- POST /api/v1/auth/login - 登录
- GET /api/v1/users/me - 获取个人信息
- PUT /api/v1/users/me - 更新个人信息
```

### 3.2 简历模块 (Resume Service)

```
职责: 简历创建、编辑、生成、导出

API:
- POST /api/v1/resumes - 创建简历
- GET /api/v1/resumes - 列表
- GET /api/v1/resumes/{id} - 详情
- PUT /api/v1/resumes/{id} - 更新
- DELETE /api/v1/resumes/{id} - 删除
- POST /api/v1/resumes/generate - AI生成简历
- GET /api/v1/resumes/{id}/export - 导出PDF
```

### 3.3 JD解析模块 (Job Description Service)

```
职责: JD解析、结构化、存储

API:
- POST /api/v1/jd/parse - 解析JD
- GET /api/v1/jd/{id} - JD详情
- GET /api/v1/jd/search - 搜索JD
```

### 3.4 匹配模块 (Match Service)

```
职责: 用户与JD匹配度计算

API:
- POST /api/v1/match - 计算匹配度
- GET /api/v1/match/history - 匹配历史
- GET /api/v1/match/recommendations - 推荐职位
```

### 3.5 对话模块 (Chat Service)

```
职责: 对话式信息采集、问答

API:
- POST /api/v1/chat/sessions - 创建会话
- POST /api/v1/chat/sessions/{id}/messages - 发送消息
- GET /api/v1/chat/sessions/{id} - 获取会话历史
```

## 4. 数据流

### 4.1 简历生成流程

```
用户输入信息 → 对话服务采集 → 用户画像构建 → 
AI生成简历 → 简历服务存储 → 用户确认 → 导出
```

### 4.2 岗位匹配流程

```
用户上传JD → JD解析服务 → 结构化数据 →
匹配服务计算 → 返回匹配度 + 差距分析 → 
生成改进建议
```

## 5. 安全设计

### 认证授权
- JWT Token 认证
- 访问令牌 + 刷新令牌
- 权限角色: admin, user

### 数据安全
- HTTPS 强制
- 敏感数据加密存储
- API 限流防刷

### 隐私合规
- 用户数据脱敏
- 数据删除机制
- 隐私政策合规
